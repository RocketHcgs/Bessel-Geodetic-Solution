# BesselGeodeticSolution
大地测量学课程设计 - 白塞尔大地主题正反算

- bessel1.m 白塞尔大地主题正算

  输入：B1 L1 A1 S

  输出：按下表顺序输出

  | 正算表格 第一列 | 正算表格 第二列 |
  | --------------- | --------------- |
  | B1              | A               |
  | L1              | B               |
  | A1              | C               |
  | S               | α               |
  | W1              | β               |
  | sinu1           | σ0              |
  | cosu1           | sin2(σ1+σ0)     |
  | sinA1           | cos2(σ1+σ0)     |
  | cosA1           | σ               |
  | sinA0           | δ               |
  | cos^2A0         | sinσ            |
  | cotσ1           | cosσ            |
  | sin2σ1          | λ               |
  | cos2σ1          | B2              |
  |                 | L2              |
  |                 | A2              |

- bessel2.m 白塞尔大地主题反算

  输入：B1 L1 B2 L2

  输出：按下表顺序输出
  
  | 反算表格 第一列 | 反算表格 第二列 |
  | --------------- | --------------- |
  | B1              | sinσ            |
  | L1              | cosσ            |
  | B2              | σ               |
  | L2              | sinA0           |
  | W1              | x               |
  | W2              | y               |
  | sinu1           | α               |
  | cosu2           | β'              |
  | sinu2           | δ               |
  | cosu2           | A               |
  | a1              | B"              |
  | b1              | C"              |
  | a2              | A1              |
  | b2              | A2              |
  | p               | S               |
  | q               |                 |
